## 📝 Deskripsi

Folder ini berisi source code utama project. Struktur folder ini dapat disesuaikan dengan kebutuhan project Anda.

## 📁 Struktur Folder yang Disarankan

```
src/
├── assets/           # Static assets (images, fonts, etc.)
├── components/       # UI components (untuk project frontend)
├── modules/          # Business logic modules
├── utils/            # Utility functions
├── services/         # API services
├── config/           # Configuration files
└── types/            # Type definitions
```

## 🚀 Cara Penggunaan

1. Tambahkan source code Anda di folder ini
2. Organisir dengan struktur yang jelas
3. Dokumentasikan setiap module
